Учет складских поставок

Описание проекта

Программа разработана в рамках производственной практики в Хромтауской районной больнице.

Проект предназначен для автоматизации учета складских поставок медицинских материалов и оборудования.

  Основные возможности

- Авторизация пользователей
- Добавление новых поставок
- Редактирование информации
- Удаление записей
- Поиск поставок
- Работа с локальной базой данных SQLite

 Используемые технологии

- Python 3
- Tkinter
- SQLite3
- SQL

Структура проекта

```
project.py           # Основной файл программы
sklad_bolnica.db     # База данных SQLite
README.md            # Описание проекта
requirements.txt     # Зависимости
```

Запуск программы

1. Установить Python 3.
2. Скачать файлы проекта.
3. Запустить:

```bash
python project.py
```
